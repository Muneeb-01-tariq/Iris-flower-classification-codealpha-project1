# Iris-flower-classification-codealpha-project1
# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project is a Machine Learning classification model that predicts the species of an Iris flower based on its sepal and petal measurements. The model was built using Python and the Scikit-learn library as part of my Data Science Internship.

The objective is to classify Iris flowers into one of the following three species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## 📂 Dataset

The project uses the **Iris.csv** dataset containing **150 flower samples**.

### Features
- SepalLengthCm
- SepalWidthCm
- PetalLengthCm
- PetalWidthCm

### Target Variable
- Species

---

## 🎯 Project Objectives

- Load and explore the Iris dataset.
- Perform data preprocessing.
- Split the dataset into training and testing sets.
- Train a Decision Tree Classifier.
